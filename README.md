Sales Data Analysis

Objective
To analyze Superstore sales data and identify trends, profitability, and regional performance.

Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

Analysis Performed
- Data Cleaning
- Monthly Sales Trend Analysis
- Profit Analysis
- Category-wise Sales
- Regional Analysis
- Correlation Heatmap

Key Insights
- Sales varied across months
- Some products contributed heavily to revenue
- Discounts negatively impacted profits
- Regional sales performance differed significantly

Conclusion
The project demonstrated exploratory data analysis and business insight generation using Python.
